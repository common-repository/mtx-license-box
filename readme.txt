== Description ==

This plugin adds a box that contains the license of the article (or any other information you want to enter).
The box can be closed with an animation effect.
The contents of the box, its color and the effect of closure can be configured in the Settings page.
This plugin has been developed as an example of a tutorial on building plugins on <a href="http://www.yourinspirationweb.com">Your Inspiration Web</a>

 
== Installation ==


1. Upload folder mtx-license-box to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Proceed to the configuration in the settings menu.
